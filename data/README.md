# Data Directory

This directory contains the dataset and related files used for training and evaluating the breast cancer classification model.

## Dataset
- **Name:** BreakHis Dataset
- **Type:** Histopathological Images
- **Classes:**
  - Benign
  - Malignant

## Directory Structure

```
data/
├── benign/
├── malignant/
├── train/
├── validation/
└── test/
```

## Note

The original BreakHis dataset is not included in this repository due to licensing and file size limitations. Please download it from the official source before training the model.
